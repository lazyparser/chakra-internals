# chakra-internals

收集和整理微软IE/Edge浏览器JS引擎ChakraCore的相关资料.

Chakra 将于 2016 年 1 月开源, 至此四大浏览器的JS引擎将全部 Open Source \o/

欢迎 `fork` & `Pull Request`!
